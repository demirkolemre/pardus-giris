Was ist Ubuntu MATE?
    |                        

Ubuntu MATE ist ein Betriebssystem -- die Kernsoftware, die Ihren Computer zum laufen bringt, ähnlich wie   Microsoft's Windows,   Apple's OS X und   Google's Chrome OS. Ubuntu MATE ist eine Distribution (Variation ) von GNU/Linux. Eine zuverlässige, sicher, fähig und moderne Computersystem dass alle anderen in der Popularität und die tatsächliche Nutzung mithalten kann.

Im Gegensatz zu anderen Betriebssystemen, ist Linux eine führender Betriebssystem der zum weitem Einsatz kommt:

  • auf den meisten der weltweit schnellsten und leistungsstärksten Supercomputer
  • auf viele (wann auch nicht alle) die Computer, die das Rückgrat des Internets bilden
  • und auf Unternehmensservern, die Stabilität und Zuverlässigkeit erfordern

Ubuntu MATE ist eine stabile, leicht zu bedienendes Betriebssystem mit einer konfigurierbaren Desktop-Umgebung. Es ist ideal für diejenigen, die das Beste aus ihren Computern wollen und eine herkömmliches Desktop- Metapher bevorzugen. Mit bescheidenen Hardware-Anforderungen, ist es geeignet für moderne Desktop-PCs und Laptops, Single-Board- Computer (wie der Raspberry Pi ) und ältere Hardware auch.

Das Betriebssystem ist sicherer und besser unterstützt als das vorinstallierte Betriebssystem auf die meisten gängigen Heim- Computer-Hardware.

Die Verwendung von Ubuntu MATE, bietet Ihnen die Freiheit, eine komplette voll funktionsfähige Betriebssystem laufen zu lassen. Ist vorkonfiguriert mit den meisten (wenn nicht alle ) Anwendungen, die Sie für Ihre tägliche Computing benötigen. Ändern Sie etwas über die Art und Weise wie es aussieht, wie es funktioniert, oder die ausgewählte Anwendungen nach Ihrem Geschmack.

Ziele

Ubuntu MATE hat eine Reihe von Führungsziele und Ziele:

  • Zugänglich für alle, unabhängig von der Sprache und körperlichen Fähigkeiten.
  • Erhöhen sowohl Ubuntu und MATE Desktop- Benutzerakzeptanz.
  • Ubuntu ist eine Alternative für Computer, die nicht stark genug sind, um einen Compositing Desktop zu treiben.
  • Erste Wahl Ubuntu -Plattform für Fern-Workstation-Lösungen (remote workstation solutions) wie LTSP und x2go.
  • Stellen Sie die glückliche Zeit von Ubuntu für Anwender wieder her, die eine herkömmliche Desktop- Metapher bevorzugen.
  • Verwenden Sie Themen und Grafiken ähnlich wie Ubuntu, so dass Ubuntu MATE Ihnen sofort vertraut ist.
  • Wenn möglich, Debian beitragen, sowohl die Debian und Ubuntu Gemeinden profitieren.
  • Software-Auswahl wird Funktionalität und Stabilität über die Leichtigkeit und Laune begünstigen.

Was ist Ubuntu MATE?

Ubuntu

Ubuntu ist einer der, wenn nicht die, größten im Einsatz Linux basierten Desktop- Betriebssysteme in der Welt. Linux ist das Herzstück von Ubuntu und macht es möglich, sichere, leistungsstarke und vielseitige Betriebssysteme zu schaffen, sowie Ubuntu und Android. Android ist jetzt in den Händen von Milliarden von Menschen auf der ganzen Welt und es ist auch von Linux betrieben.

Ubuntu ist in einer Reihe von verschiedenen Desktop-Umgebungen erhältlich, jeder kommt mit seiner eigenen Desktop-Umgebung. Ubuntu MATE nimmt die Ubuntu- Basis-Betriebssystem und fügt die MATE Desktop dazu.

MATE Desktop dazu

Wikipedia sagt, dass eine Desktop-Umgebung ist:

eine Implementierung der Desktop Metapher eines zusammengefügten Bündels von Programmen die auf einem Computer- Betriebssystem laufen, die eine gemeinsame grafische Benutzeroberfläche (GUI) teilen. Desktop- GUIs helfen dem Anwender einen einfachen Zugriff auf und Bearbeiten von Dateien.

Die MATE -Desktop implementiert wesentliche Funktionen, die Sie in einer Desktop-Umgebung erwarten würde, sowie:

Alle sind hochgradig anpassbar und über das   Kontrollzentrum verwaltete.

MATE Desktop bietet eine intuitive und attraktive Desktop-Umgebung mit traditionellen Metaphern, was bedeutet, wenn Sie überhaupt   Windows oder benutzt haben   OS X, wird es ich sehr vertraut fühlen.

Die MATE -Desktop hat eine reiche Geschichte und ist die Fortsetzung des GNOME- 2-Desktop, es war die Standard- Desktop-Umgebung auf vielen Linux und Unix-Betriebssysteme seit mehr als einem Jahrzehnt. Dies bedeutet, dass MATE Desktop erprobt, getestet und sehr zuverlässig ist.

Open Source (Frei-Quellecode)

Was Linux, Ubuntu und MATE -Desktop alle gemeinsam haben, ist, dass sie Open Source sind. Diese Software kann frei verwendet, verändert und geteilt werden von jedermann (in veränderter oder unveränderter Form). Auf den Punkt gebracht ist Ubuntu MATE im wahrsten Sinne des Wortes frei (Kostenlos).

Obwohl Sie für den Kauf einige Distributionen (Variationen) von Linux finden, die überwiegende Mehrheit sind kostenlos zur Verfügung gestellt, wie Ubuntu MATE. Open Source Software ist in einer Weise lizenziert, das jedermann erlaubt es kostenlos zu verschenken ohne weitere Bedingungen. Zum Beispiel, gibt die Lizenz jedes Mitglied des User-Community (Benutzer -Gemeinde) die Freiheit, Linux für jeden Zweck zu verwenden, zu verteilen, zu modifizieren, zu vertreiben, oder sogar das Betriebssystem zu verkaufen. Wenn Sie Änderungen vornehmen und dann Linux neu verteilen mit Ihren Änderungen, werden Sie durch die Lizenz Bedienungen, um Ihre Änderungen für die mögliche Aufnahme in zukünftige Versionen ein zu reichen. Es gibt keine Garantie, dass dies jemals passieren wird aber wenn Sie es verbessert haben, dann könnten die Änderungen in der nächsten Version (Ausgabe) von Ubuntu MATE enthalten sein.

Viele der Nutzer von Linux sind Unternehmen, die das Betriebssystem verwenden um ihre Geschäfte zu führen, oder sind sie in ihren Produkten enthalten. Viele dieser Unternehmen stellen Fehlerkorrekturen und neue Funktionen für Linux fest, wahrend sie die Software für ihre Arbeit nutzen. Diese Verbesserungen werden an der Linux -Gemeinde zurückgegeben und Ubuntu MATE wird verbessert als ein Ergebnis. Dies ist, wie wir kontinuierlich verbessern und wachsen kann, ohne dass wir von unsere Nutzer, Geld verlangen müssen.

Im Gegensatz zu Windows und OS X wird Linux nicht nur von ein Unternehmen geschaffen und unterstützt. Es wird von Intel, Redhat, Linaro, Samsung, IBM, SUSE, Texas Instruments, Google, Canonical, Oracle, AMD und Microsoft unterstützt. Über 4.000 Entwickler trugen zu Linux Entwicklung in den letzten 15 Jahren.

Egal, ob Sie ein Heimanwender von Ubuntu MATE sind, ein Ubuntu MATE Software oder Anwendungsentwickler, oder Mitarbeiter einer Organisation, die das Betriebssystem verwendet, Sie sind Mitglied der Linux und Open Source -Gemeinden und profitieren von den Anstrengungen der Entwickler, die auf Ubuntu MATE und die damit verbundenen Projekte beitragen: Linux, MATE und Ubuntu. Die Mitglieder der Gemeinschaft setzen Linux auf fast jeder Hardware ein, von der hübscheste Macbook zum billigsten Netbook, der neuesten Chromebook zu einigen sehr alten Maschinen für Windows, und von den mächtigsten Internet -Server auf den kleinsten intelligenten Thermostat (smart thermostat).
