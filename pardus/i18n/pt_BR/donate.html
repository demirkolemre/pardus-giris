Ubuntu MATE é mantido pela nossa comunidade. Suas doações ajudarão a pagar:

  • Web hosting e custos de banda.
  • Apoiando os projetos de Software Livre dos quais o Ubuntu MATE depende.
  • Eventualmente pagar por um desenvolvedor em tempo integral para o Ubuntu MATE e o MATE Desktop.

Patreon é um jeito único de fundar um projeto de Open Source. Um pagamento mensal garante sustentabilidade para o projeto Ubuntu MATE. Doadores do Patron serão recompensados com noticias exclusivas sobre o projeto, atualizações e convites para participar em videoconferências onde poderá falar diretamente com os desenvolvedores.

Se você prefere usar PayPal Então, temos opções para donar mensualemnte ou somente doar apenas uma vez finalmente aceitamos qualquer doação Bitcoin.
