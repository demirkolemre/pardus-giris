Algumas notas importantes.

Ainda que a experiência seja visualmente semelhante à edição Desktop do Ubuntu MATE, a arquitetura é muito diferente. Esta página irá destacar algumas diferenças.

Redimensionando o sistema de arquivos

Seu sistema de arquivos já foi redimensionado.

Após configurar o Ubuntu MATE pela primeira vez no seu Raspberry Pi, o sistema de arquivos estará restrito ao tamanho da imagem original, que é cerca de 3,9 GB. Utilize este botão para redimensiona-lo e ocupar todo o cartão Micro-SD.

  Redimensionar Agora

To complete the resize of your Raspberry Pi file system, please close all open windows and restart now.

  Reboot

Software Incompatível

Alguns pacotes disponíveis para o Ubuntu foram escritos e compilados apenas para a arquitetura i386 ou amd64 Como o Raspberry Pi é baseado em armv7, esses pacotes não estarão disponíveis para instalação. A depender do pacote, é possível compila-lo você mesmo ou utilizar um port existente.

Incompatível:
  • Google Chrome
  • Minecraft (Java Edition)
  • Adobe Flash
  • VirtualBox
Compatible:
  • Chromium
  • Minecraft: Pi Edition
  • Pepper Flash
  • QEMU

Atualizações

Por favor não tente atualizar seu Raspberry Pi para uma nova versão de distribuição (por exemplo de 15.04 para 15.10), pois o kernel utilizado é o not designed para isso. Este processo consumirá um longo tempo para terminar e potencialmente encherá o seu cartão SD de forma a não deixar espaço livre algum.

É mais seguro fazer um back up de todos os dados que você deseja manter e resetar o cartão com a nova imagem. Uma tentativa de atualização pode corromper o cartão SD, impedindo o sistema de iniciar ou causando sérios erros.

No entanto, você pode intalar atualizações regulares através das Atualizações de Software para os seus softwares instalados.

Atualizações de Kernel

O mesmo kernel oferecido pela fundação Raspberry Pi is used in this edition of Ubuntu MATE. As this kernel is delivered like a "firmware" blob, updates are not distributed via the Atualizações de Software ou apt-get.

Instead, to update the kernel, open a terminal and run:

sudo rpi-update

Hardware Acceleration

Currently, hardware accelerated applications are not supported unlike Raspbian. Applications that depend on OpenGL ES libraries or require the GPU will fail to start.

For playing videos, the application omxplayer will be able to do this and is pre-installed. If you are looking to play MPEG-2 or VC-1 video files then you will need MPEG-2 and/or VC-1 licenses from the Raspberry Pi Store.

Enable/Disable X11

For users who are looking to create their own headless "server" using Ubuntu MATE, there is a utility for toggling the graphical environment.

To disable X11 and login via the console:

sudo graphical disable

To enable X11 to restore the Ubuntu MATE desktop:

sudo graphical enable
Changes take effect after a reboot.
