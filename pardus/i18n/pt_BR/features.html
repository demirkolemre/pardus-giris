Funcionalidades

Moderno e Funcional

O Ubuntu MATE é um sistema operacional moderno, com uma interface de usuário atraente e fácil de entender. Seu gerenciador de atualizações mantém não apenas o sistema operacional em si, mas todos os aplicativos instalados nas versões mais atuais. É um sistema operacional mais seguro e com melhor suporte do que os sistemas que vêm pré-instalados na maioria dos computadores atualmente.

Pré-Configurado mas Flexível

Utilizar o Ubuntu MATE é uma experiência de liberdade para executar um sistema operacional funcional, pré-configurado com a maioria, se não todos os aplicativos que você precisará para uso cotidiano - ou para você modificar a aparência, a funcionalidade ou os aplicativos em uso de acordo com o seu gosto.

Segurança incorporada

O Ubuntu MATE é projetado com a segurança em mente. Ao contrário dos sistemas operacionais que se atualizam apenas uma vez por mês, o Ubuntu MATE recebe atualizações continuamente. Essas atualizações incluem correções de segurança para o Ubuntu MATE e todos os seus componentes. As atualizações de segurança para todos os aplicativos instalados também são oferecidas continuamente. Isso garante que você terá a proteção mais atual para todos os softwares do seu computador tão logo quanto disponíveis!

O Ubuntu MATE pode pegar vírus e outros softwares maliciosos... mas não pega! As atualizações rápidas e frequentes garantem que há pouquíssimas ameaças aos sistemas Linux, como o Ubuntu MATE. Elas não são mais ameaças a qualquer pessoa utilizando uma distribuição Linux moderna. O Ubuntu MATE é projetado para dificultar a instalação de vírus, root kits e outros softwares maliciosos serem instalados e executados sem a sua intervenção. Mesmo se você acidentalmente acionar um vírus, as chances são que ele tenha sido desenvolvido para o sistema Windows, não causando danos ao Ubuntu MATE.

Outra característica importante do Ubuntu MATE é que os utilizadores não são administradores do sistema por padrão. Os administradores (usuários "root") de um computador têm permissão para fazer o que quiserem, incluindo danificar o sistema.

Por exemplo, outros sistemas operacionais observam a extensão de um arquivo para determinar qual aplicativo deve abri-lo, e então tentam abri-lo imediatamente. Esse comportamento torna fácil para um intruso atacá-lo. Assim, mesmo se um programa malicioso se identifica por "Últimos desejos e Testamento.txt" o Ubuntu MATE o reconhecerá como um programa. O sistema avisará que o arquivo não é um arquivo de texto , mas um programa que será executado se você assim permitir. Para ser extra seguro, o Ubuntu MATE solicitará a senha de administrador antes que a permissão seja concedida. Isso acontecerá todas as vezes.

Aplicativos Poderosos

Enquanto o MATE Desktop oferece as interfaces de usuário essenciais para controlar e utilizar o computador, o Ubuntu MATE adiciona uma coleção de aplicativos extras para transformar o seu computador em uma estação de trabalho poderosa.

Produtividade

Firefox é um seguro, e simples browser de internet. Com uma interface familiar, fornece recursos seguros incluindo proteção de roubo de identidade online, e com uma pesquisa integrada para você obter o melhor da internet.

Thunderbird é um cliente de e-mail, RSS e um cliente de notícias cheio de funções, que torna este processo mais seguro, rápido e fácil melhor do que qualquer visto antes. Suporta diferentes contas de e-mail (POP, IMAP, Gmail), possui um simples assistente de configuração de contas, livro de endereços em um-clique, interface de guias, um filtro integrado de spam, pesquisa avançada e capacidade de indexação, e de fácil organização de e-mail por abas e pastas virtuais. Também contém funções incomparáveis.

LibreOffice é uma suíte de produtividade com todos os recursos necessários que te fornecem uma ótima alternativa ao Microsoft(R) Office; uma interface limpa com ferramentas poderosas que libertam a sua criatividade e aumentam a sua produtividade. LibreOffice contém diversos programas que o fazem o mais poderoso suíte Grátis e Código Aberto do mercado:

  • Writer, o processador de texto.
  • Calc, o aplicativo para planilhas eletrônicas.
  • Impress, o motor de apresentações.
  • Draw, o aplicativo para desenho e fluxogramas
  • Base, o banco de dados e a interface do banco de dados.
  • Math, para editar fórmulas matemáticas.

Entretenimento

Rhythmbox é um aplicativo muito simples para organizar e ouvir músicas que suporta uma enorme lista de formatos de áudio (incluindo mp3 e ogg). Originalmente inspirado no Apple iTunes, a versão atual também suporta Rádio via Internet, integração com iPod e áudio-players genéricos, gravação de Áudio CDs, playback de Áudio CDs, compartilhamento de música e Podcasts.

Shotwell is a digital photo organizer. It allows you to import photos from disk or camera, organize them in various ways, view them in full-window or fullscreen mode, and export them to share with others. It is able to manage a lot of different image formats, also including raw CR2 files.

VLC é o player de mídia do projeto VideoLAN. Ele roda MPEG, MPEG-2, MPEG-4, DivX, MOV, WMV, QuickTime, WebM, FLAC, MP3, Ogg/Vorbis, DVDs, VCDs, podcast e streaming multimídia de várias fontes da internet.

VLC can also be used as a streaming server that duplicates the stream it reads and multicasts them through the network to other clients, or serves them through HTTP.

VLC has support for on-the-fly transcoding of audio and video formats, either for broadcasting purposes or for movie format transformations.

Jogos

Nos últimos anos, Linux se tornou uma plataforma de jogos de primeira classe graças ao Valve trazendo a Steam plataforma para o Linux. No período de escrita desse texto, o Steam já possui mais de 1000 jogos indies e AAA de alta qualidade disponíveis para Linux. O Ubuntu MATE é totalmente compatível com o Steam para Linux.

While Steam is a major step forward for gaming on Linux, there are also many high quality and enjoyable Open Source games titles available for Ubuntu MATE. It doesn't matter if you like flight simulators, motor racing, first person shooters, jump and run or card games, you'll find something to keep you entertained.

Acessibilidade

Making an accessible operating system was a key priority when the Ubuntu MATE founders initially set out the goals of the project.

Ubuntu MATE is never going to be able to offer the comprehensive integration of assistive technologies that Sonar GNU/Linux provides. But Ubuntu MATE will continue to follow their lead so that Ubuntu MATE is a viable desktop solution where computer access is shared within a household or business where individual needs differ.

Integração com dispositivos móveis

Currently Ubuntu MATE has the essential mobile device support you'd expect. You can plug in a phone, media player or digital camera and the device is automatically detected and mounted. You can then access the files on it or sync its contents.
