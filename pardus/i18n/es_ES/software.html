    Reparaciones Diversas

    Esta sección contiene operaciones que pueden arreglar problemas comunes como encontrar un error al actualizar o instalar nuevo software.

    ¿Listas de Paquetes Obsoletas?

    Tus listas de repositorios pueden estar obsoletas, lo que puede ocasionar errores de No Encontrado e información desactualizada al intentar instalar versiones nuevas o más recientes del software. Este es particularmente el caso cuando Ubuntu MATE se conecta en línea por primera vez después de la instalación.

    Actualiza las listas de repositorios:
    sudo apt-get update
    Actualizar Lista de Fuentes Actualizar Paquetes Instalados

    ¿Paquetes Rotos?

    Cuando una instalación o eliminación previa fue interrumpida (por ejemplo, debido a un fallo de corriente o una pérdida de conexión a Internet), no puede ser añadido o eliminado más software sin reconfigurar apropiadamente estos paquetes rotos. Si es necesario, puede que necesites solucionar dependencias e instalar cualesquiera paquetes ausentes con el fin de ejecutar el software correctamente.

    Configurar paquetes que fueron desempaquetados pero no configurados:
    sudo dpkg --configure -a
    Descargar e instalar dependencias rotas:
    sudo apt-get --fix-broken install
    Configure Paquetes Interrumpidos Solucionar Paquetes Rotos

    Además, listados abajo tienes comandos equivalentes de la terminal que de todas formas aparecen en el Actualizador de Software.

    Actualizar todos los paquetes que tengan una nueva versión disponible:
    sudo apt-get dist-upgrade
    Actualizar a una nueva versión de Ubuntu MATE:
    sudo do-release-upgrade
