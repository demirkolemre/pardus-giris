Características

Moderno y Completamente Equipado

Ubuntu MATE es un sistema operativo moderno para ordenador, con una interfaz de usuario atractiva y fácil de entender. Su gestor de actualizaciones mantiene no solo al sistema operativo en sí, sino a todas sus aplicaciones instaladas, actualizados a la versión actual. El sistema operativo es más seguro, y está mejor soportado que sistemas operativos que vienen preinstalados en la mayoría de hardware de computación doméstico de hoy.

Preconfigurado Pero Flexible

Usar Ubuntu MATE te proporcionará la libertad para ejecutar un sistema opertaivo completo y repleto de funcionalidades, preconfigurado con la mayoría de (si no todas) las aplicaciones que necesitarás para tu informática diaria. Cambia cualquier cosa de su apariencia, la forma en que trabaja, o las aplicaciones que ejecuta para adaptarse a tu gusto.

Seguridad Incorporada

Ubunu MATE está diseñado con la seguridad en mente. A diferencia de sistemas operativos que se actualizan solo una vez al mes, Ubuntu MATE recibe actualizaciones continuamente. Las actualizaciones incluyen parches de seguridad para Ubuntu MATE y todos sus componentes. Actualizaciones de seguridad para todas las aplicaciones instaladas son también provistas en el mismo plan. ¡Esto te garantiza lo último en protección para todo el software de tu ordenador - tan pronto como esté disponible!

Ubuntu MATE puede tener virus y otras infecciones... pero no los tiene. Actualizaciones rápidas y puntuales aseguran que hay muy pocas, si acaso alguna, amenazas a sistemas Linux como Ubuntu MATE que persisten en lo salvaje. En realidad, ha habido muy pocas infecciones "públicas" en los últimos 10 años que puedan afectar a Ubuntu MATE. Ya no son una amenaza para nadie que instale o use una distribución Linux moderna hoy. Ubuntu MATE está diseñado para dificultar a virus, rootkits y otro malware el ser instalados y ejecutados sin intervención consciente por tu parte, la del usuario. Incluso si accidentalmente provocas una infección, hay posibilidades de que esté diseñada para atacar a Windows y no haga mucho, si acaso algún daño a Ubuntu MATE.

Otra característica de seguridad significativa de Ubuntu MATE es que sus usuarios no son administradores por defecto. Los administradores (usuarios "root") de cualquier sistema de ordenador tienen permisos para hacer cualquier cosa que quieran, incluyendo la habilidad para dañar al sistema.

Por ejemplo, otros sistemas operativos miran el nombre de un fichero para determinar que programa debería abrirlo, entonces intenta inmediatamente abrirlo. Ese diseño facilita a un intruso el atacar un ordenador. Ubuntu MATE abre un fichero basándose en que fichero es, no en su nombre. Así que aun si un programa maligno disfraza su identidad usando el nombre "Últimas Voluntades y Testamento.txt" Ubuntu MATE reconocerá el fichero como un programa. El sistema pondrá una advertencia como que el fichero no es un fichero de texto, pero que es un programa que se ejecutará si le das permiso para continuar. Para ser extra seguro, Ubuntu MATE exige que proporciones tu contraseña de administrador antes de que el permiso sea concedido. Cada vez.

Aplicaciones Potentes

Mientras que el Escritorio MATE proporciona las interfaces de usuario esenciales para controlar y usar un ordenador, Ubuntu MATE añade una colección de aplicaciones adicionales para convertir a tu ordenador en una estación de trabajo verdaderamente potente.

Productividad

Firefox cumple con una navegación web fácil y segura. Una interfaz de usuario familiar, características de seguridad mejoradas incluyendo protección contra usurpación de identidad en línea, y una búsqueda integrada para permitirte sacar el máximo partido a la web.

Thunderbird es un cliente de email, RSS y grupo de noticias completamente equipado que hace al correo electrónico más seguro, rápido y fácil que nunca antes. Soporta diferentes cuentas de correo (POP, IMAP, Gmail), tiene un asistente de configuración de cuentas de correo simple, agenda de acceso en un-click, una interfaz por pestañas, un filtro de correo basura inteligente integrado, capacidades avanzadas de búsqueda e indexación, y ofrece una fácil organización de correos con carpetas etiquetables y virtuales. Presenta también una extensibilidad sin rival.

LibreOffice es una suite de productividad ofimática completamente equipada que provee un reemplazo casi inmediato a Microsoft(R) Office; Su limpia interfaz y sus potentes herramientas te permiten desatar tu creatividad y aumentar tu productividad. LibreOffice incorpora varias aplicaciones que lo convierten en la suite Ofimática Libre y de Código Abierto más potente del mercado:

  • Writer, el procesador de texto.
  • Calc, la aplicación para hojas de cálculo.
  • Impress, el motor de presentaciones
  • Draw, la aplicación de dibujo y creación de diagramas de flujo
  • Base, el motor de base de datos y la interfaz.
  • Math, para edición matemática

Entretenimiento

Rhythmbox es un programa de reproducción y gestión de música muy fácil de usar que soporta un rango amplio de formatos de audio (incluyendo mp3 y ogg). Originalmente inspirado en iTunes de Apple, la versión actual también soporta Radio en Internet, integración para iPod y soporte para reproductores portátiles genéricos de audio, grabación de Audio en CD, reproducción de CD de Audio, compartición de música, y Podcasts.

Shotwell is un organizador de fotos digitales. Te permite importar fotos desde disco o cámara, las organiza de varias maneras, las visualiza en modo ventana o pantalla completa, y las exporta para compartirlas con otros. Es capaz de gestionar un montón de formatos de imagen diferentes, incluyendo también ficheros raw CR2.

VLC es el reproductor multimedia del proyecto VideoLAN. Reproduce MPEG, MPEG-2, MPEG-4, DivX, MOV, WMV, QuickTime, WebM, FLAC, MP3, Ogg/Vorbis files, DVDs, VCDs, podcasts, y transmisiones multimedia de varias fuentes de red.

VLC también puede ser utilizado como un servidor de transferencia continua que duplica la transmisión que los lee y realiza multidifusión a través de la red a otros clientes, o los atiende a través de HTTP.

VLC tiene soporte para transcodificación al vuelo de formatos de audio y video, bien para propósitos de difusión o bien para conversiones de formatos de película.

... Y mucho más!

Naturalemente también encontrarás un cortafuegos, una aplicación para copias de seguridad, gestión de escáner e impresora para documentos/fotos todo incluido en Ubuntu MATE. Y esto es solo el principio. Las sección de Software incluye las mejores aplicaciones de su clase para la mayoría de tareas y un instalador de un click para una "App Store" que te dará acceso a miles de aplicaciones adecuadas para más o menos cualquier actividad profesional o recreativa.

Juegos

En el par de años Linux se ha convertido en una plataforma de videojuegos de primera clase gracias a Valve por traer la plataforma Steam a Linux. Mientras esto se escribe, Steam tiene más de 1.000 títulos independientes y AAA de alta calidad disponibles para Linux. Ubuntu MATE es completamente compatible con Steam para Linux.

Mientras que Steam es un gran paso adelante para los videojuegos en Linux, hay también muchos títulos de juegos de Código Abierto disfrutables y de alta calidad para Ubuntu MATE. No importa si te gustan los simuladores de vuelo, las carreras de motor, los disparos en primera persona, saltar y correr o los juegos de cartas, encontrarás algo que te mantendrá entretenido.

Accesibilidad

Hacer un sistema operativo accesible fue una clave prioritaria cuando los fundadores de Ubuntu MATE inicialmente presentaron las metas del proyecto.

Ubuntu MATE nunca va a ser capaz de ofrecer la integración completa de tecnologías de asistencia que Sonar GNU/Linux proporciona. Pero Ubuntu MATE seguirá su liderato para que Ubuntu MATE sea una solución de escritorio viable donde el acceso al ordenador esté compartido, dentro de un hogar o un negocio donde las necesidades individuales difieren.

Integración Móvil

Actualmente Ubuntu MATE tiene el soporte esencial de dispositivo móvil que esperarías. Puedes conectar tu móvil, reproductor multimedia o cámara digital y el dispositivo es automáticamente detectado y montado. Después puedes acceder a los ficheros en él o sincronizar su contenido.
