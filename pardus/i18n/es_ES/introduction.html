¿Qué es Ubuntu MATE?
    |                        

Ubuntu MATE es un sistema operativo - la base de software que hace funcionar a tu ordenador, similar a   Windows de Microsoft,   OS X de Apple y   Chrome OS de Google. Ubuntu MATE es una distribución (variación) de GNU/Linux. Un sistema de ordenador fiable, seguro, capaz y moderno que rivaliza con todos los otros en popularidad y uso real.

A diferencia de otros sistemas operativos, Linux es muy usado:

  • en la mayoría de los superordenadores más rápidos y poderosos del mundo
  • en muchos (si no en la mayoría) de los ordenadores que constituyen la columna vertebral de Internet
  • y en servidores corporativos que requiren estabilidad y fiabilidad

Ubuntu MATE es un sistema operativo estable y de fácil uso con un entorno de escritorio configurable. Es ideal para aquellos que quieran sacar el máximo provecho de sus ordenadores y prefieren una metáfora de escritorio tradicional. Con requerimientos de hardware modestos, es apropiado para ordenadores sobremesa y portátiles moderno, ordenadores de una sola placa (como la Raspberry Pi) y hardware antiguo por igual.

El sistema operativo es más seguro, y está mejor respaldado que el sistema operativo preinstalado en la mayoría del hardware de computación doméstico de hoy.

Usar Ubuntu MATE te proporcionará la libertad para ejecutar un sistema opertaivo completo y repleto de funcionalidades, preconfigurado con la mayoría de (si no todas) las aplicaciones que necesitarás para tu informática diaria. Cambia cualquier cosa de su apariencia, la forma en que trabaja, o las aplicaciones que ejecuta para adaptarse a tu gusto.

Objetivos

Ubuntu MATE tiene un número de objetivos y metas guía:

  • Accesible para todos, independientemente de sus habilidades físicas o del lenguaje.
  • Aumentar la adopción de Ubuntu y el escritorio MATE por parte de los usuarios.
  • Alternativa a Ubuntu para ordenadores que no son lo suficientemente potentes para ejecutar un escritorio compuesto.
  • Primera elección como plataforma Ubuntu para soluciones de estaciones de trabajo remotas como LTSP y X2Go.
  • Recrear tiempos añorados de Ubuntu para usuarios que prefieren una metáfora de escritorio tradicional.
  • Usar temas y trabajos artísticos similares a Ubuntu para que Ubuntu MATE resulte inmediatamente familiar.
  • Cuando sea posible, contribuir con Debian para que así ambas comunidades de Debian y Ubuntu se beneficien.
  • La selección de software favorecerá la funcionalidad y la estabalidad por encima de la rapidez y la extravagancia.

¿Qué es Ubuntu MATE?

Ubuntu

Ubuntu es uno de los, si no el, más desplegados Linux sistemas operativos para escritorio del mundo. Linux se encuentra en el corazón de Ubuntu y hace posible crear sistemas operativos seguros, potentes y versátiles, como Ubuntu y Android. Android está ahora en las manos de miles de millones de persona alrededor del mundo y funciona con Linux.

Ubuntu está disponible en una serie de diferentes sabores, cada cual con su propio entorno de escritorio. Ubuntu MATE toma el sistema operativo Ubuntu base y añade el Escritorio MATE.

Escritorio MATE

Wikipedia dice que un Entorno de Escritorio es:

una implementación de la metáfora de escritorio compuesta por un lote de programas que se ejecutan por encima de un sistema operativo de ordenador, que comparten una interfaz gráfica de usuario (GUI) común. Las GUIs de escritorio ayudan al usuario a acceder y editar ficheros fácilmente.

El Escritorio MATE implementa características esenciales que esperarías de un entorno de escritorio, como:

Buscador de Ficheros Calculadora Gestor de Archivos Visor de Imágenes Visor de Documentos Monitor del Sistema Terminal
caja galculator engrampa eom atril mate-system-moniotr mate-terminal
Todas ellas altamente personalizable y gestionadas a través del   Centro de Control.

El Escritorio MATE proporciona un entorno de escritorio intuitivo y atractivo usando metáforas tradicionales, lo que significa que si alguna vez has usado   Windows o   OS X, lo encontrarás muy familiar.

El Escritorio MATE tiene una rica historia y es la continuación del escritorio GNOME 2, que fue el escritorio por defecto de muchos sistemas operativos Linux y Unix durante aproximadamente una década.

Código Abierto

Lo que todos, Linux, Ubuntu y el Escritorio MATE, tienen en común es que son Código Abierto. Este software puede ser usado, cambiado y compartido (en formas modificadas o sin modificar) libremente por cualquiera. En pocas palabras Ubuntu MATE es libre, en el sentido más verdadero de la palabra.

Although you will find some distributions (variations) of Linux for purchase, the vast majority are provided free of charge, like Ubuntu MATE. The distribution is licensed in a way that allows anyone to give it away for free, with no strings attached. For example, the licence gives any member of the user community the freedom to use Linux for any purpose, to distribute, modify, redistribute, or even sell the operating system. If you do modify and then redistribute Linux with your modifications, you are required by the licence to submit your modifications for possible inclusion into future versions. There is no guarantee that this will ever happen, but if you have made it better, then your changes just might be included in the next release of Ubuntu MATE.

Muchos de los usuarios de Linux son corporaciones que usan el sistema operativo para hacer funcionar sus negocios, o incluirlo entre sus productos. Muchas de estas corporaciones aportan parches y nuevas características pues usan el software para sus negocios. Estas mejoras son devueltas a la comunidad Linux y como resultado Ubuntu MATE mejora. Así es como podemos mejorar y crecer continuamente sin tener que cobrar dinero a nuestros usuarios.

A diferencia de Windows y OS X, Linux no está creado ni respaldado por una sola compañía. Está respaldado por Intel, RedHat, Linaro, Samsung, IBM, SUSE, Texas Instruments, Google, Canonical, Oracle, AMD y Microsoft. Aproximadamente 4.000 desarrolladores contribuyeron con Linux durante los últimos 15 años.

Seas un usuario doméstico de Ubuntu MATE, un desarrollador de software o aplicaciones para Ubuntu MATE, o un empleado en una organización que utilice el sistema operativo, eres un miembro de las comunidades de Linux y de Código Abierto y te beneficias de los esfuerzos de los desarrolladores quienes contribuyen con Ubuntu MATE y sus proyectos relacionados: Linux, MATE y Ubuntu. Los miembros de la comunidad ejecutan Linux en casi cualquier hardware, desde el Macbook más bonito al netbook más barato, desde el Chromebook más nuevo a algunas máquinas muy viejas diseñadas para Windows, y desde los servidores de Internet más potentes al termostato inteligente más pequeño.
