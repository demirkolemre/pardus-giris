Información sobre Raspberry Pi

Unas pocas notas importantes.

Mientras que la experiencia es visualmente similar a la edición de escritorio de Ubuntu MATE, la arquitectura subyacente es muy diferente. Esta página destacará unas pocas de esas diferencias.

Software Incompatible

Algunos paquetes disponibles para Ubuntu han sido únicamente escritos y compilados para la i386 o amd64 arquitectura. Como la Raspberry Pi está basada en armv7, estos paquetes no estarán disponibles para instalar. Dependiendo del paquete, puede ser posible compilar o utilizar un puerto existente.

Incompatibles:
  • Google Chrome
  • Minecraft (Java Edition)
  • Adobe Flash
  • VirtualBox
Compatible:
  • Chromium
  • Minecraft: Pi Edition
  • Pepper Flash
  • QEMU

Actualizaciones

Por favor no intentes actualizar tu Raspberry Pi a una versión más reciente de la distribución (por ejemplo, desde 15.04 a 15.10), ya que el núcleo subyacente será not designed el que lo haga. Este proceso

Es más seguro respaldar todos los datos que desea mantener y volver a grabar la tarjeta con la nueva imagen. El intento de actualización puede dañar la tarjeta SD, evitar que su instalación arranque, o provocar fallos graves.

Puedes, sin embargo, instalar actualizaciones normales a través de la Actualización de software herramienta para tu software instalado.

Actualizaciones del Núcleo

El mismo núcleo proporcionado por la fundación Raspberry Pi se usa en esta edición de Ubuntu MATE. Como este núcleo es distribuido como un blob "firmware", sus actualizaciones no son distruidas a través del Actualización de software o apt-get.

En su lugar, para actualizar el núcleo, abre una terminal y ejecuta:

sudo rpi-update

Aceleración por Hardware

Actualmente, las aplicaciones aceleradas por hardware no están soportadas diferente a Raspbian. Las aplicaciones que dependen de las bibliotecas OpenGL ES o que requieren la GPU no se podrán iniciar.

Para reproducir vídeos, la aplicación omxplayer será capaz de hacerlo y viene preinstalada. Si buscas reproducir archivos de vídeo MPEG-2 o VC-1 entonces necesitarás las licencias MPEG-2 y/o VC-1 de la Tienda Raspberry Pi

Habilita/Deshabilita X11

Para usuarios que estén buscando crear sus propios "servidores" sin interfaz gráfica usando Ubuntu MATE, hay una herramienta para alternar con el entorno gráfico.

Para deshabilitar X11 e iniciar sesión a través de la consola:

sudo graphical disable

Para habilitar X11 y volver al escritorio de Ubuntu MATE:

sudo graphical enable
Los cambios tienen efecto después de un reinicio.
