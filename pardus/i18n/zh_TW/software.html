    這個章節包含可修復一般問題的操作，當您升級或者安裝新軟體時您是否遭遇到問題?

    過期套件清單?

    您的軟體庫清單可能已過期，可能造成 找不到 錯誤和 過期的版本 在首次安裝完成之後，當Ubuntu MATE連上線時會試圖安裝新版軟體或者更新

    更新軟體庫清單
    sudo apt-get update
    更新來源檔清單 更新已安裝的套件

    缺少套件?

    當之前的安裝被中斷或移除 (例如，因為失去電力供應或是網路連線), 沒有適當的調整設定這些損毀的套件，許多軟體將不能被新增或者移除。為了使軟體可以正常的運作，您也需要解決相依性及安裝所有缺少的套件。

    調整未調整的未打包套件
    sudo dpkg --configure -a
    下載並安裝缺少的相依性套件
    sudo apt-get --fix-broken install
    調整中斷的套件 解決缺少的套件

    除此之外，下列等同於終端機指令 軟體更新管理員.

    升級所有有新版本的套件:
    sudo apt-get dist-upgrade
    升級至新發行的Ubunutu MATE:
    sudo do-release-upgrade
