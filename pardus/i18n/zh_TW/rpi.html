一些重要的公告

雖然可能跟桌面版的Ubuntu MATE經驗很類似，但裡面的微架構非常的不同。此頁面將會指出這些不同點。

重新調整檔案系統

After setting up Ubuntu MATE for the first time on your Raspberry Pi, the file system will be restricted to the size of the original image, which is about 3.9 GB. Use this button to resize and fill the entire Micro-SD card.

  立即重新調整

To complete the resize of your Raspberry Pi file system, please close all open windows and restart now.

  Reboot

不相容的軟體

某些針對Ubuntu的套件只被編譯給 i386 或 amd64 微架構。 Raspberry Pi是基於 armv7, these packages will not be available to install. Depending on the package, it may be possible to compile these yourself or use an existing port.

不相容的:
  • Google Chrome
  • Minecraft (Java Edition)
  • Adobe Flash
  • VirtualBox
Compatible:
  • Chromium
  • Minecraft: Pi Edition
  • Pepper Flash
  • QEMU

升級

請 不要 想要 升級 您的Raspberry Pi升級至更新的發行版本(例如，從15.04到15.10)，當目前的內核是 not designed 繼續。這個過程需要很長的時間來完成並且有可能塞爆您的SD卡

It is safer to back up all of your data you wish to keep and re-flash the card with the new image. Attempting to upgrade may corrupt the SD card, prevent your installation from booting, or cause severe glitches.

然而您還是可以安裝 一般更新 經由 軟體更新管理員 提供給已安裝軟體的常用工具

內核更新

相同的內核由以下提供 Raspberry Pi基金會 被用在這個版本的Ubuntu MATE。只要這個內核像是"軔體"blob，更新檔就不會經由以下發佈 軟體更新管理員 或 apt-get.

相反地，如要更新內核，請打開終端機並且執行:

sudo rpi-update

硬體加速

目前硬體加速的應用程式 不支援 unlike Raspbian. Applications that depend on OpenGL ES libraries or require the GPU will fail to start.

這個應用程式提供播放影片 omxplayer 將可以繼續且已預先安裝。如果您在尋找MPEG-2或VC-1影片檔案 您需要MPEG-2和/或者 VC-1憑證從 Raspberry Pi商店

開啟/關閉X11

對於想要自行建立無標頭的使用者 伺服器 Ubuntu MATE提供開關圖形環境的小工具

關閉X11，使用CLI登入

sudo graphical disable

開啟X11繼續Ubuntu MATE桌面

sudo graphical enable
重新開機變更才會生效
