    Corrections Diverses

    Cette section contient des opérations pouvant résoudre les principaux problèmes dans le cas où vous rencontreriez une erreur à la mise à jour ou l'installation de nouveaux logiciels.

    Listes de Paquets obsolètes?

    Vos listes de dépôts sont peu-être trop anciennes, et peuvent causer des erreurs du type Élément Non Trouvé ainsi que des informations de version périmées au moment d'installer de nouveaux logiciels ou de nouvelles version de celui-ci, ce qui est particulièrement vrai au moment où Ubuntu MATE se connecte au réseau pour la première fois après l'installation.

    Mettre à jour les listes de dépôts:
    sudo apt-get update
    Mettre à jour les listes des Sources Mettre à jour les Paquets Installés

    Paquets Cassés ?

    Quand une installation ou désinstallation a été intérrompue (par exemple à cause d'une coupure de courant ou d'une perte de connexion internet), aucun autre logiciel ne peut être ajouté ou retiré sans re-configurer proprement ces paquets cassés. Si nécessaire, il se peut que vous deviez aussi résoudre des problèmes de dépendances et installer tout paquet manquant pour que le ou les logiciels fonctionnent correctement.

    Configurer les paquets ouverts et non configurés :
    sudo dpkg --configure -a
    Télécharger et installer les dépendances cassées :
    sudo apt-get --fix-broken install
    Configurer les Paquets en Suspend Résoudre les Problèmes de Paquets Cassés

    De plus, voici une liste ci-dessous des commandes équivalentes pour le terminal qui en temps normal apparaissent dans Mises à Jour Logiciels.

    Mettre à jour tous les paquets ayant une nouvelle version disponible :
    sudo apt-get dist-upgrade
    Mettre le système à jour vers une nouvelle publication de Ubuntu MATE :
    sudo do-release-upgrade

    Les Logiciels Propriétaires sont des logiciels dont les droits sont possédés par un individu ou une société. Leur utilisation peut être soumise à une restriction (par exemple un accord de licence), et leur code source non-accessible.

    Les Gratuiciels et Logiciels Open Source donnent à l'utilisateur la liberté de partager, étudier et modifier leur code sans restriction. Concrètement, GPL est un exemple de licence "Logiciel Libre".
